\begin{frame}[fragile]\frametitle{Syntonising RF SOCs on White-Rabbit}

\begin{itemize}
	\item White-Rabbit provides timing for systems of interest, generally by feeding them clocks derivied from the main WR clock.
	\item Assumes that no frequency approximation is done in the system of interest (wrong in the AD9361 case),
	\item Can't grant phase synchronisation because of PLL indeterminism.
\end{itemize}

\begin{frame}[fragile]\frametitle{PLL Indeterminism}
\includegraphics[width=\linewidth]{pll_indeterminism_fig.pdf}

\begin{minipage}{.5\linewidth}
{\footnotesize
\begin{itemize}
	\item PLL multiplying by 1.5, division by 2 yields 2 possible phase positions.
	\item Sampling these 2 possible signals at the same time would give 2 patterns: \texttt{10} or \texttt{01}
	\item A PLL with divisor N gives N patterns each N bits long. With a LUT we know what position we are in and we can then correct it: https://wiki.confluence.nikhef.nl/wiki/spaces/WRNIK/pages/934182916/Lock+Sweep
	\item By simulating the sampling of the patterns, we can automaticaly generate the LUT for given frequencies
\end{itemize}}
\end{minipage}
\begin{minipage}{.49\linewidth}
\includegraphics[width=.\linewidth]{plt_pll_lock_possibilities.pdf}

A PLL dividing by 5 yeilds 5 possibilities
\end{minipage}
\end{frame}

\begin{frame}[fragile]\frametitle{Auxiliary Locksweep}

\begin{itemize}
	\item Auxiliary soft PLLs cost an extra VCO, and offer fine phase control over a 62.5 MHz feedback clock.
	\item This solves the frequency approximation issue,
	\item Using the phase sampler and fine phase control, we can correct the PLL indeterminism.
\end{itemize}

\end{frame}

\begin{frame}[fragile]\frametitle{SDR synchronization demonstration}

\begin{minipage}[t]{\linewidth}
\begin{minipage}{.49\linewidth}
\begin{itemize}
\item Feed the AD9361 input with a BPSK modulated carrier carrying a pseudo-random sequence (2.5 Mchips/s)
\item Cross-correlation of the IQ stream to identify delay with a resolution improvement
over the sampling rate equal to the SNR
\item Repeat after restarting WR to assess proper phase alignement of the recorded IQ stream
(ADC clock synchronization)
\item However, the receiver local oscillators remain free running: random RF phase between
SDR receivers despite fine phase alignement of the ADC samples (problem for electronic beam/null
stearing).
\end{itemize}
